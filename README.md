# **EN LANGUAGE**

**Espada - Automated Telegram Account/Channel Takedown System**

This project is a Python script for automatically sending complaints about accounts and channels in Telegram through official support email addresses.

# Warning

**This script is created for educational purposes only. The author is not responsible for any misuse of this software. Use only for testing your own systems or for legitimate purposes.**

# Features
> 1. Account takedown (3 types of complaints)
> 2. Channel takedown (7 types of violations)
> 3. Multi-account system - use of unlimited email accounts
> 4. Random delays between sends to simulate human behavior

# Installation on Termux

```pkg update && pkg upgrade```

```pkg intall git```

```git clone https://https://github.com/espadawo/espada-snos```

```cd espada-snos```

```python espada.py```
