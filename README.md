**Espada - Автоматизированная система для сноса аккаунтов/каналов Telegram**

Этот проект представляет собой Python - скрипт для автоматической отправки жалоб на аккаунты и каналы

**Предупреждение**

Этот скрипт создан исключительно в образовательных целях. Автор не несет ответсвенности за любое неправомерное использование данного программного обеспечения. Используйте только для тестирования в собственных целлях или в законных целях.

**Функционал**
1. Снос аккаунтов (3 типа жалоб)
2. Снос каналов (7 типов нарушений) 
3. Мультаккаунтная система - использование не ограниченного количества почт
4. Случайные задержки между отправками для имитации человеческого поведения
