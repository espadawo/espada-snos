# Espada

# **RU LANGUAGE**

**Espada - Автоматизированная система для сноса аккаунтов/каналов Telegram**

Этот проект представляет собой Python - скрипт для автоматической отправки жалоб на аккаунты и каналы

# Предупреждение

**Этот скрипт создан исключительно в образовательных целях. Автор не несет ответсвенности за любое неправомерное использование данного программного обеспечения. Используйте только для тестирования в собственных целлях или в законных целях.**

# Функционал
> 1. Снос аккаунтов (3 типа жалоб)
> 2. Снос каналов (7 типов нарушений) 
> 3. Мультаккаунтная система - использование не ограниченного количества почт
> 4. Случайные задержки между отправками для имитации человеческого поведения

# **EN LANGUAGE**

**Espada - Automated Telegram Account/Channel Takedown System**

This project is a Python script for automatically sending complaints about accounts and channels in Telegram through official support email addresses.

# Warning

**This script is created for educational purposes only. The author is not responsible for any misuse of this software. Use only for testing your own systems or for legitimate purposes.**

# Features
> 1. Account takedown (3 types of complaints)
> 2. Channel takedown (7 types of violations)
> 3. Multi-account system - use of unlimited email accounts
> 4. Random delays between sends to simulate human behavior
